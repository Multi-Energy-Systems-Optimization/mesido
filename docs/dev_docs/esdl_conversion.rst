ESDL to MESDIO conversion
=========================

This section shows all the convert functions available to translate an ESDL asset to an asset in MESIDO. These convert
functions take the attributes from the ESDL asset, optionally do some calculations with them, after which they are
provided as modifiers to the MESIDO asset in the component library.
